About HandWriting Generation

DOMAIN: Machine Learning

  • SUBDOMAIN: Recurrent Neural Network

  • A recurrent neural network (RNN) is a class of artificial neural networks where connections between nodes form a directed graph along a temporal sequence.

  • This allows it to exhibit temporal dynamic behavior. Unlike feedforward neural networks, RNNs can use their internal state (memory) to process sequences of inputs.

  • Deep recurrent neural networks specifically Deep LSTM cells can be used along with a Mixture Density Network to generate artificial handwriting data.

  • Using this model we can generate random handwriting styles which are being hallucinated by the model

HGeneration

This main objective of this project is to shows, how Long Short-term Memory recurrent neural networks can be used to generate complex sequences with long-range structure, simply by predicting one data point at a time.
